# truthlens-ai
“TruthLens AI is a deep learning–powered system that detects whether an image is real or AI-generated, helping improve trust in digital media.”

## Features

- Detects whether an image is **real or AI-generated**  
- Provides a **confidence score** for each prediction  
- Built using **deep learning (CNN)** for high accuracy   
- Can be extended to **deepfake video detection** in the future
